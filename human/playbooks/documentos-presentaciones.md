# Documentos → presentaciones

Generado: 2026-06-23T16:56:54.761958+00:00

## Objetivo

pasar texto/informes a slides editables.

## Stack sugerido

- `markitdown`: utilidad Python de Microsoft que convierte archivos complejos (PDF, Word, Excel, PowerPoint, HTML, imágenes, audio) a Markdown limpio optimizado para que los LLMs lo consuman.
- `ppt-master`: generador que transforma documentos de texto en presentaciones PowerPoint (.pptx) editables nativas, listas para abrir y ajustar en Office.
- `revealjs`: framework veterano para crear presentaciones (slides) en HTML, con transiciones, temas, fragmentos y modo orador integrados.
