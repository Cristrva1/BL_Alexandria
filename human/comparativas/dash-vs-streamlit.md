# dash vs streamlit

Generado: 2026-06-24T18:45:16.466468+00:00

| Criterio | dash | streamlit |
|---|---|---|
| Uso real | framework para construir aplicaciones analíticas interactivas en Python combinando gráficos Plotly con componentes de UI reactivos. | framework para levantar apps internas y prototipos de datos desde Python con solo unas líneas de script, sin tocar HTML ni JS. |
| Elegir si | quieres dashboards en Python | quieres una UI de datos al vuelo |
| Evitar si | prefieres prototipos ultrarrápidos ([streamlit](#-streamlit)). | necesitas una app productiva ([dash](#-dash)). |
| Instalacion | reference_only | reference_only |

## Regla practica

Elige `dash` si su caso de uso encaja mejor ahora. Deja `streamlit` como alternativa, salvo que el proyecto necesite explicitamente ambos.
