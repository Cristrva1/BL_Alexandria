| Criterio | videofy_minimal | remotion |
|---|---|---|
| Uso real | herramienta minimalista (de Schibsted) que convierte contenido textual en videos cortos de forma local, sin pipeline complejo. | framework para crear videos por código usando React, definiendo cada frame como un componente y renderizando a MP4. |
| Elegir si | quieres reels simples ya | generas video a escala con React |

| Instalacion | reference_only | reference_only |

## Regla practica
