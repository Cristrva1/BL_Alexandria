# ppt-master vs revealjs

Generado: 2026-06-23T16:56:54.761958+00:00

| Criterio | ppt-master | reveal.js |
|---|---|---|
| Uso real | generador que transforma documentos de texto en presentaciones PowerPoint (.pptx) editables nativas, listas para abrir y ajustar en Office. | framework veterano para crear presentaciones (slides) en HTML, con transiciones, temas, fragmentos y modo orador integrados. |
| Elegir si | quieres PPTX editable | quieres slides web |
| Evitar si | prefieres slides web ([reveal.js](#-revealjs)). | necesitas PPTX editable en Office ([ppt-master](#-ppt-master)). |
| Instalacion | reference_only | reference_only |

## Regla practica

Elige `ppt-master` si su caso de uso encaja mejor ahora. Deja `revealjs` como alternativa, salvo que el proyecto necesite explicitamente ambos.
