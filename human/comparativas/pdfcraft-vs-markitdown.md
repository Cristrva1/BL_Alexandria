# pdfcraft vs markitdown

Generado: 2026-06-25T03:39:27.698706+00:00

| Criterio | pdfcraft | markitdown |
|---|---|---|
| Uso real | conjunto de herramientas PDF gratuitas, privadas y basadas en navegador para combinar, dividir, comprimir y convertir archivos sin instalar nada. | utilidad Python de Microsoft que convierte archivos complejos (PDF, Word, Excel, PowerPoint, HTML, imágenes, audio) a Markdown limpio optimizado para que los LLMs lo consuman. |
| Elegir si | manipulas PDFs con privacidad | alimentas LLMs con documentos |

| Instalacion | reference_only | local_project |

## Regla practica
