## `gsap`

**Stack:** JavaScript
**Repo:** https://github.com/greensock/GSAP

## `penpot`
role=app · exec=local · setup=heavy · mcp=False · prov=['mcp']

**Qué es:** plataforma open-source de diseño y prototipado colaborativo, pensada para equipos de producto y diseño que quieren soberanía sobre sus archivos.
**Stack:** stack web self-host, Docker
**Repo:** https://github.com/penpot/penpot

**Instalación** [~]: `git clone https://github.com/penpot/penpot && cd penpot && docker compose up -d`
_Requiere Docker; el compose puede variar — revisa docker-compose.yml._

**Elige si:** quieres diseño abierto y soberano
**Evita si:** necesitas generación por IA ([open-design](#-open-design)).
**Combina con:** `designmd`, `plasmic`
**Alternativas (elige una):** `open-design`
